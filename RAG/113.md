# A-MEM: 에이전트를 위한 살아있는 기억

> **"RAG는 저장소일 뿐 기억이 아니다. A-MEM은 스스로 진화하는 기억 네트워크를 제안합니다."**

## 1. 개요

기존 RAG는 정적인 검색 도구에 불과합니다. **A-MEM (Agentic Memory)**은 독일의 지식 관리법인 **제텔카스텐(Zettelkasten)**에서 영감을 받아, 기억을 독립적인 노트로 만들고 스스로 연결하는 시스템입니다.

## 2. 3단계 프로세스

1.  **Note Construction**: LLM이 정보를 요약하고 태그를 달아 구조화된 노트를 생성합니다.
2.  **Link Generation**: 새로운 기억과 기존 기억 사이의 의미적 연결고리를 찾아 자동으로 링크를 생성합니다.
3.  **Memory Evolution**: 새로운 경험이 들어오면 기존 기억을 업데이트하고 더 정교하게 발전시킵니다.

## 3. 성능

*   **Multi-Hop**: 기존 모델 대비 2배 이상의 성능 향상.
*   **효율성**: 토큰 사용량을 1/6 수준(16,910 -> 2,520)으로 줄이면서도 성능을 유지했습니다.

## 4. 결론

A-MEM은 에이전트가 긴 대화 속에서도 맥락을 놓치지 않고, 경험을 통해 성장할 수 있는 '살아있는 기억'을 구현했습니다.
