# Agentic Reasoning: 생각의 도구를 쓰는 에이전트

> **"LLM 혼자 끙끙대지 마라. 검색, 코딩, 마인드맵을 도구로 써서 박사급 문제를 푼다."**

## 1. 개요

단순한 CoT(Chain of Thought)를 넘어, 외부 도구와 상호작용하며 추론하는 **Agentic Reasoning** 프레임워크입니다.

## 2. 3대장 에이전트

1.  **Mind Map**: 정보를 그래프로 구조화하여 논리적 관계를 추적합니다.
2.  **Web Search Agent**: 실시간으로 최신 정보를 찾아옵니다.
3.  **Code Execution Agent**: 복잡한 계산이나 시뮬레이션을 코드로 실행합니다.

## 3. 성과

*   **GPQA (박사급 과학 문제)**: OpenAI o1 모델과 동급 이상의 정확도를 기록했습니다.
*   **Deep Research**: 의학, 법률 등 전문 분야 연구 과제에서 기존 모델보다 우수한 성능을 입증했습니다.

## 4. 결론

LLM을 '지식 저장소'가 아닌 '연구 수행자'로 재정의한 연구입니다.
