# 인공지능에서의 중요도는 데이터 90%, 에이전트 9%, 모델 1%입니다.

> **최근 화려한 에이전트와 모델 소식들에 가려져 이 본질을 잊고 지내는 것 같습니다.** 인공지능에서의 중요도는 데이터 90%, 에이전트 9%, 모델 1%입니다.

데이터란 단순한 학습용 데이터가 아닙니다.

조직이나 개인이 했던 것들을 기록하고, 어떤 문제가 있었고 지금까지 이어지는지 추적하며, 어떻게 그 문제를 해결했었는지 혹은 왜 아직도 문제를 해결하지 못했는지 기록을 남기는 것.

동시에 이를 찾기 쉽게 정리해 두는 것, 그것이 데이터입니다.

이런 데이터가 없다면 우리는 아무것도 이룰 수 없습니다. 당장 Claude Code나 OpenClaw 따위를 안 써봐도 됩니다.

본인들이 잘하는 일을 발전시키고, 모든 것을 기록하고 저장해놓고 정리하는 것이 가장 중요합니다.
