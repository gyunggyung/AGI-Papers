# PISCO: RAG를 위한 초고효율 압축

> **"문서를 16배 압축했는데 성능이 똑같다고? 사전 학습 없이 가능한 RAG 전용 압축 기술."**

## 1. 문제: 길어지는 입력, 비싸지는 비용

RAG 시스템은 검색된 문서가 많을수록 추론 비용이 기하급수적으로 증가합니다. 기존 압축 기술은 정보 손실이 크거나, 별도의 모델 학습이 필요했습니다.

## 2. PISCO의 해법

**PISCO (Pretty Simple Compression)**는 **Sequence-level Knowledge Distillation (SKD)**을 활용해 사전 학습 없이도 문서를 압축합니다.

*   **LoRA 활용**: 원본 문서를 LLM의 LoRA Adapter를 통해 작은 '메모리 토큰'으로 변환합니다.
*   **Data-free**: 별도의 학습 데이터 없이, 원본 모델이 스스로 압축된 표현을 학습하도록 만듭니다.

## 3. 결과

*   **압축률**: 최대 **16배** 압축.
*   **속도**: 추론 속도 **5.7배** 향상.
*   **정확도**: 원본 대비 성능 저하 0~3% 수준. (기존 압축 기법은 8% 이상 저하)

## 4. 결론

PISCO는 기업용 RAG 시스템의 비용 절감과 속도 향상을 위한 현실적이고 강력한 솔루션입니다.
