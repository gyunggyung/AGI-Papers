# Gemini가 선정한 AI 논문 및 인사이트 10선

작성 중인 여러 글들 중 Gemini가 선정한 주요 논문 및 인사이트 10선을 소개합니다.

1. **코딩 에이전트의 역설 (Evaluating AGENTS)**
에이전트에게 전체 리포지토리 컨텍스트를 제공하는 것이 오히려 정보 과부하를 유발해 성공률을 떨어뜨린다는 연구.

2. **16GB 로컬 에이전트 아키텍처 (Zero-Swap Tiny-MoA)**
16GB RAM 환경에서 ModernBERT 라우터와 350M 다이내믹 LoRA 스와핑을 활용해 I/O 지연을 0으로 수렴시킨 다중 에이전트 설계.

3. **LFM2-24B 터미널 에이전트 구축**
H100 1대에서 FP8 LoRA SFT 후 실환경 강화학습(RL)을 적용해 최소 비용으로 터미널 에이전트를 구축하는 훈련 가이드.

4. **다국어 소형 모델 확장 'Tiny Aya'**
합성 데이터 융합과 지역 특화 병합 기술을 통해 3B 모델 크기로 70개 국어 번역 성능을 달성한 엣지 아키텍처.

5. **디퓨전 LLM과 보안성 (Mercury 2)**
초당 1,009 토큰을 생성하는 디퓨전 방식 LLM의 파괴적 혁신과 인페인팅 기반 보안 공격 취약점 분석.

6. **ModernBERT 라우터 효율성 분석**
CPU 8비트 환경의 연산 한계를 돌파한 인코더 기반 라우터의 설계(Unpadding 및 Alternating Attention) 벤치마킹.

7. **다중 추론 루프 보상 최적화 (RLTT)**
최종 출력뿐 아니라 추론 루프의 모든 중간 궤적에 가중합 보상을 할당하여 에이전트 통제력을 극대화하는 알고리즘 메커니즘.

8. **환각 해결을 위한 'PretrainRL'**
포스트 트레이닝의 한계를 넘어, 사전 학습 단계에 빔 서치와 DPO를 결합해 근본적인 지식망을 재정렬하는 강화학습 방법론.

9. **희소 메모리 파인튜닝 (파괴적 망각 방어)**
FFN 레이어를 메모리 슬롯으로 교체해 신규 지식만 타겟팅함으로써, 파인튜닝 시 기존 지식의 망각률을 대폭 방어한 지속 학습 혁신.

10. **친노동자적 AI 패러다임**
AI의 일자리 대체 심화가 알고리즘의 문제가 아닌 경영진의 맹목적인 '자동화(비용 절감) 선호' 구조에 기인함을 파헤친 NBER 논문.
