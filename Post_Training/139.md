title: "LLM이 스스로 가설을 세우는 방식: 지식 그래프 기반 강화학습"
---

LLM은 할루시네이션을 일으킵니다. 특히 법률이나 의학 그리고 과학 연구나 신소재 개발에서는 인공지능이 지어낸 그럴싸한 거짓말이 치명적이죠. LLM에게 텍스트로 생각(CoT)을 쓰게 두지 말고, 추적 가능한 마인드맵(지식 그래프)을 강제로 그리게 만들면 어떨까요?

MIT와 미국 오크리지 국립연구소의 논문 "Graph-Native Reinforcement Learning Enables Traceable Scientific Hypothesis Generation through Conceptual Recombination"은 LLM이 가설을 세우는 새로운 방식을 제시했습니다. 그리고 제가 직접 이 논문의 코드를 뜯어보고 다양한 백본 모델로 실험하며 알게 된, 현실도 정리해 드립니다.

1. 생각의 과정을 기계가 읽는 지도로 만들다

기존 LLM은 생각의 과정이 투명하지 않은 블랙박스였습니다. 하지만 이 논문은 LLM이 질문을 받으면 '브레인스토밍 -> 관계 스케치 -> JSON 지식 그래프 작성 -> 패턴 추출 -> 가설 합성'이라는 5단계 과정을 거치도록 강제합니다.

 - 압도적 투명성: 기존 모델 대비 전반적인 성능이 40~65%나 증가했고, 특히 생각의 경로를 검증하는 추적 가능성 지표가 92%까지 증가했습니다. LLM이 내부적으로 컴퓨터가 읽을 수 있는 JSON 지도를 직접 짜면서 논리를 전개하기 때문에, 헛소리를 하려야 할 수가 없게 뼈대를 잡아버린 겁니다.

2. 장식용 그래프를 막는 가혹한 정보 보틀넥

LLM이 대충 그럴싸한 그래프를 그리는 꼼수(Reward Hacking)를 어떻게 막았을까요?

 - 스파르타식 보상: 학습 시 LLM이 뱉어낸 줄글 답변을 다 지워버립니다. 오직 LLM이 만든 JSON 그래프 하나만 판독기에 던져서 "이 지도만 보고 원래 정답을 복원해봐!"라고 시킵니다. 그래프 안에 진짜 핵심 인과관계 정보가 꽉꽉 들어차 있지 않으면 살아남지 못하게 강화학습(GRPO)을 시킨 셈이죠.

3. 연산을 늘리면 창의적 도약이 폭발한다

LLM이 질문에 답할 때마다 뱉어낸 이 작은 그래프 조각들을 몽땅 모아 하나의 거대 지식망으로 합치게 해봤습니다.

 - 놀라운 지식의 재조합: 컴퓨팅 자원(추론 시간)을 더 투입하자, LLM이 엉뚱한 헛소리(환각)를 하는 대신 한계 돌파를 보여줍니다. 예를 들어 '개미 떼의 군집 지능'과 '나노 조립'이라는 전혀 무관해 보이는 두 개념을 융합해 "스스로 상처로 몰려가 치유하는 신소재"라는 혁신적인 가설을 스스로 세우는 식입니다. 

 - 100% 의도된 창의성: 이는 우연히 두 단어가 섞인 에러가 아닙니다. AI가 이렇게 이종 학문 간의 다리를 놓은 패턴을 분석해 보니, 우연으로는 절대 불가능한 통계적 수치(일반 유의미성 기준의 수십 배)를 기록했습니다. 억지로 시키지 않아도 수천 번이나 융합적 가설을 척척 만들어내는, 진정한 자율형 창조 파트너의 탄생입니다. 

4. API 의존성의 한계

논문은 학습과 평가에 GPT, Grok, Claude 같은 상용 API를 동기식으로 수만 번 호출하는 것을 쿨하게 가정합니다. 하지만 API 비용을 감당하는 것은 쉽지 않습니다.

저는 이 병목을 뚫기 위해 로컬 BGE 임베딩과 오프라인 비동기 파일 큐(GLM52-as-judge)를 구축해 우회해서 테스트를 해봤습니다. 그러나 생각보다 재현이 어려워서 추가 실험은 포기했습니다.

마치며

이 논문은 SFT 없이 10K 데이터로 강화학습(ORPO+GRPO) 단 두 단계만으로 LLM의 사고방식을 바꾼 흥미로운 연구입니다. CoT보다 지식 그래프를 실시간으로 만드는 방식이 확실히 더 매력적으로 보입니다.

Qwen3(8B, 1.7B)와 Llama 3.2(3B)에서 해당 작업이 가능하다는 것을 멋지게 증명했습니다. 하지만 소형 모델들은 엄격한 형식을 지키는 것이 불가능하니 API 채점에 크게 의존하는 학습과 평가를 할 수밖에 없고, 이 점은 이해는 되지만 살짝 아쉽습니다.
