# VideoRAG: 영상을 읽는 RAG

> **"텍스트만 검색하는 시대는 갔다. 초장편 비디오의 맥락까지 이해하는 멀티모달 RAG."**

## 1. 개요

기존 RAG는 텍스트 중심이라 비디오 데이터를 다루기 어려웠습니다. **VideoRAG**는 텍스트, 오디오, 영상을 모두 이해하여 초장편 비디오에서도 정확한 정보를 찾아냅니다.

## 2. 핵심 기술

*   **이중 채널 아키텍처**: 텍스트 지식 그래프와 시각적 특징을 동시에 인코딩하여 비디오 간의 의미적 연결을 파악합니다.
*   **크로스 비디오 연결**: 단편 영상이 아니라, 여러 비디오에 걸친 긴 맥락을 연결하여 지식 네트워크를 구축합니다.

## 3. 성능

*   **LongerVideos 벤치마크**: 160개 이상의 비디오(134시간 분량)에서 실험.
*   **비교**: GraphRAG 대비 5~7% 높은 검색 정확도를 기록했습니다.

## 4. 활용

강의 요약, 다큐멘터리 분석, CCTV 영상 검색 등 텍스트만으로는 불가능했던 영역에서 RAG의 새로운 가능성을 열었습니다.
