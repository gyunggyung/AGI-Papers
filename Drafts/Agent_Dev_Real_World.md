# 에이전트 개발은 현실 세계와 얼마나 동떨어져 있는가?

> **벤치마크 점수만으로 자화자찬하던 에이전트 개발이, 실제 노동 수요와 자본이 어디에 있는지 실증 데이터로 일침을 맞다.**

## 1. 기존의 한계

수학이나 코딩 같은 실험실 수준의 제한된 벤치마크 점수만으로 에이전트의 성능을 평가하고, "인간을 대체할 수 있다"고 주장했습니다. 하지만 실제 비즈니스에서 인간의 노동이 집중되는 영역과 AI 에이전트가 개발되는 영역 사이에는 큰 괴리가 있었습니다.

## 2. 새로운 접근: 실증 데이터 기반 분석

이 논문은 실제 인류의 노동 수요와 자본이 어디에 몰려 있는지를 O*NET 등 실증 데이터로 분석합니다.

*   **괴리 분석**: 현재 에이전트 벤치마크는 코딩·수학에 편중되어 있지만, 실제 노동 시장의 가치는 의사소통, 관리, 창의적 판단 등 비정형 업무에 집중되어 있습니다.
*   **뼈아픈 지적**: 에이전트 개발 방향이 현실 세계의 비즈니스 니즈와 심각하게 괴리되어 있음을 정량적으로 증명합니다.
*   **방향 제시**: 벤치마크가 아닌, 실제 경제적 가치 창출 능력으로 에이전트를 평가해야 합니다.
