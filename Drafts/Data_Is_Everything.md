# 데이터가 90%다: AI 시대의 가장 중요한 습관

## 핵심 주장

> **AI 중요도: 데이터 90% · 에이전트 9% · 모델 1%**

최신 모델이나 에이전트를 쓰느냐가 아니라, **데이터를 저장하고 정리하는 습관**이 AI 시대의 핵심 경쟁력.

## 데이터란 무엇인가?

별거 아니다. **그저 기록하고 저장하는 것**:
- 고객 사용 패턴
- 해결된 문제가 **왜** 잘 됐는지
- 해결 안 되는 문제는 **무엇이 있고 왜** 안 되는지
- 고객 문의 사항과 불만

→ 이것을 AI가 잘 읽을 수 있도록 정리하는 것이 **중요도의 90%**.

## 핵심 인사이트

### 적은 노력, 최대 성과
- 데이터 정리는 **생각보다 적은 노력**으로 가장 큰 성과를 낸다
- txt, md, SQL — 형식은 상관없다. **저장만 해두면** 된다

### 도구보다 데이터
- Claude Code, CoWork, OpenClaw, ChatGPT — **안 써도 상관없다**
- 하지만 **데이터는 반드시 저장**해야 한다
- 인류의 80% 이상이 AI를 한 번도 안 써봤지만, AI는 결국 쓰기 편해질 것

### 해결 가능성의 분기점

| 상태 | 결과 |
|------|------|
| 데이터가 저장·정리되어 있음 | 문제는 **결국 해결된다** |
| 데이터가 없거나 정리 불가 | 문제는 **영원히 해결되지 않는다** |

## 결론

> "데이터가 없으면 세상을 지배할 인공지능도 그저 **헛소리 생성기**일 뿐이다."

기록하라. 정리하라. 그것이 이 시대에서 가장 중요한 습관이다.
