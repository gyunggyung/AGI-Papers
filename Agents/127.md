title: "Neural Procedural Memory (NPM): 모델의 행동 오류를 고치는 뇌 신호 조종"
---

우리는 자전거 타는 법이나 그릇을 씻는 순서를 매번 머릿속에서 글로 읽으면서 실행하지 않고, 몸에 밴 직관으로 자연스럽게 움직이죠. LLM도 CPU로 1~2초 만에 뇌 신경을 직접 건드려서 이러한 절차적 직관을 심어줄 수 있다고 합니다.

"그릇을 찬장에 넣으세요"라고 AI에게 시켰더니, 설거지도 안 한 더러운 그릇을 그대로 집어넣어 버립니다. 글자는 분명 읽고 이해했는데, 막상 연속적인 행동을 할 때는 중간 단계를 통째로 건너뛰는 현상. 

최근 AI 에이전트를 개발해 본 사람이라면 누구나 마주하는 고질적인 병, 바로 말과 행동의 불일치(Text-Action Disconnect)입니다.

그동안 우리는 에이전트가 실수를 하면 "먼저 그릇을 씻고 찬장에 넣어라"라며 프롬프트 창에 구구절절 긴 워크플로우 잔소리를 적어주었습니다. 하지만 이 방식은 문맥 창(Context Window)을 낭비해 비용과 지연 시간을 늘릴 뿐, 행동 오류를 근본적으로 고치지 못했습니다.

최근 중국 연구진(CAS-IA, BAAI)이 발표한 "Neural Procedural Memory (NPM)"는 이 병목을 효율적으로 해결합니다. 잔소리를 읽히는 대신, 모델의 뇌(신경망)를 미세한 전기 신호로 툭 쳐서 올바른 행동을 유도하는 기술입니다.

1. 말 대신 뇌 세포 직접 조종하기 (Training-free)

이 기술은 SFT나 RL 훈련 과정이 없고, 오직 수학적 연산과 뇌 신호 조종(Activation Steering)만으로 작동합니다.

 - 실패와 성공의 뇌 신호 대조: 에이전트의 과거 기록 중 성공할 때의 뇌 세포 신호(Hidden States)에서 실패할 때의 신호를 뺍니다. 그리고 "어느 방향으로 뉴런 신호를 틀어야 성공하는지" 그 차이(조종 벡터)를 수학적으로 추출해 둡니다.

 - 실시간 수액 주입: 새로운 임무가 주어지면, 모델이 다음 행동을 고민하며 단어를 뱉는 바로 그 순간 신경망 중간 레이어(Qwen3 기준 17~19번 레이어)에 이 벡터를 실시간으로 더해줍니다.

글자를 단 한 줄도 읽지 않았는데, 모델은 무의식적으로 "반복 루프 돌던 바보짓은 피하고 다음 단계로 넘어가야겠구나" 하고 행동을 고쳐 잡습니다.

2. 성능 향상을 만드는 뇌 속의 앰프와 밀당

단순히 느낌만 좋아지는 게 아닙니다. 모델 내부를 분해해 보면, 조종 벡터는 매우 정교하게 모델의 행동 뉴런을 통제하고 있습니다.

 - 나쁜 버릇 억제: 모델이 헤매고 있을 때 튀어나오는 의미 없는 반복 관찰 뉴런이나 마음대로 끝내버리는 조기 종료 뉴런의 볼륨을 강제로 눌러 차단합니다.

 - 좋은 직관 증폭: 반대로 작업을 완수하는 데 꼭 필요한 체계적인 검색이나 최종 물건 배치 뉴런의 활성화 볼륨은 확 키워줍니다.

3. 45KB의 기적과 4배 빠른 속도

이 기술의 진정한 가치는 가성비에 있습니다.

 - 초경량 연산: 추출된 조종 벡터의 크기는 한 작업당 고작 45킬로바이트(KB)에 불과합니다. 수천만 원짜리 서버 GPU 대신, 일반 CPU 환경에서도 단 1~2초 만에 주성분 분석(PCA) 연산으로 추출할 수 있습니다.

 - 지연 시간 소멸: 프롬프트 텍스트 토큰을 전혀 쓰지 않기 때문에, 첫 글자가 나오는 속도(Prefill Latency)는 71.09ms로 기존 텍스트 RAG 방식(279.89ms) 대비 약 4배나 빠릅니다.

4. 소형 모델(sLLM)에서 터지는 시너지

이 기술이 가장 큰 위력을 발휘하는 최적의 구간은 4B에서 8B 규모의 소형 모델 진영입니다. 기초 지식은 충분하지만 집중력이 약해 자꾸 삼천포로 빠지는 소형 모델의 뒷덜미를 잡고 강제로 올바른 경로만 걷게 만듭니다.

 - 가상 홈 미션(ALFWorld, 에이전트가 방을 돌아다니며 물건을 찾고 씻고 데우는 등 복잡한 멀티스텝 평가)에서 39.55%에 불과했던 Qwen3-8B 모델의 성공률이 NPM 벡터 주입만으로 56.72%까지 증가했습니다.

 - 쇼핑몰 탐색(WebShop, 유저의 까다로운 조건에 맞는 상품을 검색하고 옵션을 골라 결제해야 하는 환경 평가)에서는 촘촘한 텍스트 지침을 줄 때(45.73점)보다, 텍스트 없이 순수하게 NPM 뇌 신호 조종만 했을 때(48.00점) 오히려 점수가 더 높게 나오는 기현상이 벌어졌습니다.

마치며

물론 잘 안 될 때도 있습니다. 가끔 작업의 난이도를 인공지능이 잘못 판단하여 엉뚱한 조종 벡터를 주입하는 경계선 케이스(Boundary cases)가 발생합니다. 

실제로 ScienceWorld 환경의 Qwen3-4B 모델 실험에서는 잘못된 신호 간섭으로 인해 점수가 18.40점에서 18.20점으로 소폭 깎이는 역효과가 관찰되었습니다.

그럼에도 전반적인 성능 향상을 보이는 것은 물론, 모델이 이미 처리할 수 있는 잠재력이 뇌 속에 존재한다는 점을 간파하고, 무거운 학습 비용을 단 1원도 쓰지 않은 채 가성비 좋게 직관을 깨워주었습니다. (다만, 데이터 생성을 위해서 모델 실행을 할 때는 GPU가 필요합니다)

Activation Steering 분야도 점점 발전하고 있습니다. 검열 해제, tool-calling 성능 상승을 지나 이제는 현실에 대한 이해를 늘려가고 있습니다. 다음은 무엇일지 기대되네요.
