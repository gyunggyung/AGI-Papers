# 로컬 터미널 에이전트 CLI 개발 및 LFM2-8B-Terminal-SFT 튜닝

Pau Labarta Bajo 님의 조언을 받아서 로컬 터미널 에이전트 CLI를 만들었습니다. 그리고 Unsloth AI 공식 LFM2-8B-A1B의 1달 다운로드 수보다, 제가 이를 터미널 환경에 최적화되게 튜닝한 LFM2-8B-Terminal-SFT-Unsloth (A1B)의 6일 다운로드 수가 2.6배 이상 많아졌습니다.

아직 SFT 학습도 1/4밖에 못했고, Multi-turn 학습도 RL 학습도 하지 못했기에, 아주 복잡한 작업은 못합니다. 그래도 꽤 신기하고 쓸 만합니다. 

RAM 16GB, CPU, 윈도우(WSL2) 환경에서도 잘 돌아가고, A1B다 보니 추론 속도가 꽤 빠릅니다. 다만, 출력하는 텍스트가 꽤 길어서 시간이 상당히 걸립니다. 

우리 업계의 최고 전문가인 Maxime Labonne 님께서도 잘 만들었다고 하니, 한번 써보시길 바랍니다. (영어로만 쓸 수 있으니, 구글 번역기를 활용해 보세요)

### 🚀 실행 방법

아래 방법으로 바로 실행할 수 있습니다. (참고로 리눅스 환경에서만 사용 가능합니다. 윈도우 사용자는 WSL2 (Ubuntu) 환경에서 사용해 주세요.) 

```bash
git clone https://github.com/gyunggyung/liquid-cli
cd liquid-cli
uv run liquid ask "Check current CPU and Memory usage"
```

그리고 활용 예시 명령어는 다음과 같습니다.

```bash
uv run liquid ask "Find all .json files and extract lines containing '5e8' from them."
```

아마 해당 Github는 월요일쯤 Liquid AI cookbook에 올라갈 것 같습니다.

이 외에도 진행 중인 작업들이 상당히 많은데, 어서 공개할 수 있었으면 좋겠습니다.
