저는 처음 들어보는 API를 써서 기능을 만드니 진짜 대단하네요. 그리고 얘가 만든 코드 퀄리티 자체가 매우 좋습니다. Gemini 3 Pro를 그냥 쓰는 것보다 훨씬 잘 만드는 것 같아서 재밌네요. 

더 신기한 사실은 명령을 수행할 때마다, 결과물이 좋아지고 있습니다. 영상에서 보여드린 웹 결과물은 역순으로 보여드린 겁니다. 메모리 기능을 아직 따로 구현하지 않았는데, Gemini CLI 덕인지, 저도 모르게 메모리 기능을 구현한 건지 모르겠네요.


제가 무언가를 만들면서, 내가 뭘 만든거지? 라는 생각이 드는 경험은 처음이라서 신기하네요. 스스로 코딩을 해서 새로운 가능을 만드는 기능은 일단 제가 코드를 다 이해한 후에 추가해야겠습니다. 아마 평일 중에 만들 수 있지 않을까 싶습니다.

안 쓰는 우분투 노트북이 있는데, 포멧하고 그냥 보안도 무시하고 마음대로 해보라고 해서 어떤 결과물이 나오는지 확인해봐야겠습니다. 물론 아무것도 못하고 끝날 가능성이 더 큽니다. 그런데 만약에 무언가를 만들어내면 재밌겠네요.
