# LOTaD: 에이전트는 어떻게 일을 나눠야 할까?

> **"사람이 일일이 일을 쪼개주지 않아도, 에이전트가 스스로 최적의 분업 방식을 찾아낸다."**

## 1. 개요

다중 에이전트 강화학습(MARL)에서 가장 어려운 건 "누가 무엇을 할지" 정하는 것입니다. **LOTaD (Learning Optimal Task Decompositions)**는 이 과정을 자동화했습니다.

## 2. 작동 원리

1.  **후보 생성**: 랜덤하게 여러 가지 작업 분해 방식을 만듭니다.
2.  **동시 학습**: 에이전트들이 모든 분해 방식에 대해 동시에 학습하며 경험을 공유합니다.
3.  **UCB 선택**: 학습 과정에서 가장 성과가 좋은 분업 방식을 **UCB(Upper Confidence Bound)** 알고리즘으로 선택합니다.

## 3. 결과

*   **Repairs Task**: 기존 대비 **38.7%** 성능 향상.
*   **Cooperative Buttons**: **26.4%** 빠른 학습 속도.

## 4. 결론

사람의 개입 없이도 팀워크를 최적화하는 LOTaD는 복잡한 협업 에이전트 시스템의 자동화를 앞당길 것입니다.
