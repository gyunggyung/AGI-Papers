# ADGR: 스스로 지도를 그리는 에이전트

> **"지식 그래프를 미리 만들어둘 필요 없다. 에이전트가 필요할 때마다 개념을 연결하고 확장한다."**

## 1. 개요

기존 지식 그래프는 정적입니다. **ADGR (Agentic Deep Graph Reasoning)**은 LLM이 스스로 개념을 생성하고 연결하여 **동적인 지식 네트워크**를 구축합니다.

## 2. 프로세스

1.  **Initialization**: 질문을 받고 초기 개념 네트워크를 구축합니다.
2.  **Recursive Expansion**: LLM이 새로운 개념을 찾아내어 기존 그래프에 연결합니다. (Hub와 Bridge 형성)
3.  **Self-Refinement**: 너무 복잡해지지 않게 불필요한 연결을 정리(Pruning)합니다.

## 3. 결과

*   **GPQA**: 기존 대비 **42.8%** 성능 향상.
*   **효율성**: 기존 LLM보다 연산량을 **55%** 줄이면서도 더 뛰어난 추론 능력을 보여주었습니다.

## 4. 결론

ADGR은 AI가 단순히 검색하는 것을 넘어, 인간처럼 탐구하고 지식을 확장하는 **자기 조직적(Self-Organizing)** 학습을 가능하게 합니다.
