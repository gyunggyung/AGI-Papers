# MetaChain: 말만 하면 만들어지는 에이전트

> **"코딩 몰라도 됩니다. 자연어로 설명하면 에이전트와 워크플로우를 알아서 짜줍니다."**

## 1. 개요

LangChain, AutoGen은 어렵습니다. **MetaChain**은 **Zero-code** 방식으로 누구나 멀티 에이전트 시스템을 구축할 수 있게 해줍니다.

## 2. 기능

*   **자동 생성**: "금융 분석해줘"라고 하면 웹 검색, 파일 분석, 계산기 에이전트를 알아서 조합합니다.
*   **Self-Managing**: 업로드된 파일을 자동으로 벡터 DB로 변환하고 관리합니다.
*   **Self-Play**: 사용자의 피드백을 받아 에이전트 구성을 스스로 최적화합니다.

## 3. 결과

*   **GAIA 벤치마크**: 기존 SOTA 모델들을 제치고 최상위 성능을 기록했습니다.
*   **RAG**: 기존 LangChain RAG보다 10% 이상 높은 정확도를 보였습니다.

## 4. 결론

에이전트 개발의 민주화를 이끌 중요한 도구입니다. 이제 기획력만 있으면 누구나 AI 시스템을 만들 수 있습니다.
